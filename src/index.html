Data Information

Enter the total size of your data in gigabytes. This helps determine the storage requirements.
Enter the number of batch cycles your data processing requires. This affects the compute resources needed.
Enter the number of tables in your database. Note: This parameter has a minimal impact on SKU size.
Select whether Copilot is enabled for your workload. Enabling Copilot can increase the SKU size due to additional compute requirements.
Select how often your data is refreshed. More frequent refreshes require more compute resources.
Select the period for which data is retained. Longer retention periods may require more storage.
Enter the number of users accessing the system. More users can increase the compute and storage requirements.
Select the complexity level of your data. More complex data requires more compute resources for processing.
